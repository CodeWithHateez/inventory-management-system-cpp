# 📦 Inventory Management System (C++)

A console-based Inventory Management System built in **C++** that demonstrates core **Data Structures & Algorithms (DSA)** concepts with real-world inventory operations like product management, searching, sorting, and file handling.

---

## ✨ Features

### 🛒 Product Management

* ➕ Add new products
* 📝 Edit existing products
* ❌ Remove products
* 🔍 Search product by ID
* 📋 Display all products

---

### 📊 Inventory Operations

* 🏷️ Filter products by category
* 💰 Filter by price range
* ⚠️ Low stock alerts system
* 💾 Save & load inventory using file handling

---

### 🧠 DSA Concepts Implemented

* 🔗 Hash Table → Fast product storage & retrieval
* 🌳 Binary Search Tree (BST) → Structured product organization
* 🔎 Binary Search → Efficient name-based search
* 🔢 Bubble Sort → Sort products by price
* 📚 Merge Sort → Sort products by name

---

## 🛠️ Tech Stack

* 💻 C++
* 🧱 Object-Oriented Programming (OOP)
* 📂 File Handling
* ⚙️ Data Structures & Algorithms

---

## 📁 Project Structure

* `Product Class` → Product details & methods
* `HashTable Class` → Fast lookup storage
* `BST Class` → Sorted data structure
* `Inventory Class` → Core system logic
* `main()` → Menu-driven interface
